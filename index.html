Materiales Utilizados

📦 Cartón

Se utiliza para construir la estructura principal de las maquetas gracias a su resistencia y facilidad para cortar.

🥤 Botellas Plásticas

Ideales para fabricar árboles, edificios, depósitos de agua y elementos decorativos.

📰 Papel y Periódico

Se emplea para crear detalles, revestimientos y papel maché.

🪵 Palitos de Helado

Perfectos para construir puentes, cercas, techos y estructuras resistentes.

🧻 Tubos de Cartón

Los tubos de papel higiénico o servilletas sirven para fabricar columnas, torres y chimeneas.

🥫 Latas de Aluminio

Se reutilizan para representar tanques, postes y otros elementos metálicos.

🥚 Cartones de Huevos

Útiles para crear montañas, jardines, tejados y decoraciones con relieve.

🧴 Tapas Plásticas

Se utilizan como ruedas, ventanas, señales, mesas o elementos decorativos.

🧵 Tela y Retazos

Permiten elaborar cortinas, banderas, techos o detalles decorativos.

🌿 Materiales Naturales

Hojas secas, ramas, piedras pequeñas y semillas ayudan a representar jardines y paisajes.

🎨 Pintura y Pegamento

Se emplean para dar color, unir las piezas y mejorar el acabado final de la maqueta.

📐 Herramientas Básicas

Tijeras, regla, lápiz y silicona son indispensables para medir, cortar y ensamblar los materiales.
